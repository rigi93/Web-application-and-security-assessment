How to run the code and Get site running:

1) Download the file and unzip it.

2) Install Python 2.7.12

3) Install Django 1.9

4) On terminal type: python manage.py runserver

5) In the browser http://127.0.0.1:8000/Rigapp/

Functionalities:

Users can upload images along with optional caption and the site will upload to the site. Pagination is implemented to provide only 10 results at a time. The images are sorted chronologically
